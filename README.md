# beforebuy (Vercel-ready)

- Stack: Vite + React + TypeScript
- Entry: `index.html` -> `src/main.tsx` -> `src/App.tsx`
- Your Figma Make code copied into `src/components`, `src/imports`, `src/lib`

## Local (optional)
npm i
npm run dev

## Build
npm run build

## Deploy (Vercel)
- Import this repo in Vercel
- Framework: Vite
- Build command: npm run build
- Output directory: dist
